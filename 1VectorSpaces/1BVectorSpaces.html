Vector Addition and Scalar Multiplication

1.19 definition: addition, scalar multiplication

  • An addition on a set V is a function that assigns and element \(u + v \in V\) to each pair of elements \(u,v \in V\)
  • A scalar multiple on a set V is a function that assigns an element \(\lambda v \in V\) to each \(\lambda \in \mathbb{F}\) and each \(v \in V\).

note:

Defn 1.19 ensures closure in \(V\)

Vectors

definition: vector

Elements of a vector space are called vectors or points.

note:

  • \(\mathbb{F}^n\) is a vector space over \(\mathbb{F}\) (with properties satisfying the definition of vector space given below).

  • The point or vector of \(\mathbb{F}^n\) is given by: \((x_1,\dots,x_n)\) where \(x_k \in \mathbb{F} \text{ for } k = 1,..,n\)

Vector Space

1.20 definition: vector space

A vector space is a set \(V\) along with an addition and scalar multiplication on \(V\) such that the following properties hold.

1. commutativity
\(u+v = v+ u\) for all \(u,v \in V\)

2. associativity
\((u+v)+w = v+(u+w)\) and \((ab)v = a(bv)\) for all \(u,v,w \in V\) and for all \(a,b \in F\).

3. additive identity
There exists an element \(0 \in V\) such that \(v + 0 = v\) for all \(v \in V\).

4. additive inverse
For every \(v \in V\) there exists \(w \in V\) such that \(v+w=0\).

5. multiplicative identity
\(1v = v\) for all \(v \in V\)

6. distributive properties
\(a(u+v) = au+av\) and \((a+b)v = av+bv\) for all \(a,b \in F\) and \(u,v \in V\).


Note: difference between a Field and a Vector Space

A field has a multiplicative inverse, i.e. for evey \(\alpha \in \mathbb{F}\) there exists a unique \(\beta \in \mathbb{F}\) such that \(\alpha\beta = 1\). A vector space does not have this condition - also: multiplication between vectors is not defined. But a vector space does have an additive inverse - that is for \(v \in V\) there exist a unique inverse \(-v\) such that \(v + (-v) = 0\).

Examples: Vector spaces:

\(\mathbb{R}\) - real vector space over \(\mathbb{R}\)
\(\mathbb{C}\) - complex vector space over \(\mathbb{C}\)
\(\mathbb{F}^n\) - finite dimensional vector space over \(\mathbb{F}\)
\(\mathbb{F}^\infty\) - infinite vector space over \(\mathbb{F}\)
\(\mathbb{F}^S\) - the set of functions from \(S\) to \(F\)

Set of functions \(\mathbb{F}^S\) is a vector space.

notation: \(\mathbb{F}^S\)

  • If \(S\) is a set, then \(F^S\) denotes the set of functions from \(S\) to \(\mathbb{F}\).
  • For \(f,g \in \mathbb{F^S}\), the sum \(f+g \in \mathbb{F^S}\) is the function defined by \[(f+g)(x) = f(x)+g(x)\] for all \(x \in S\).
  • For \(\lambda \in \mathbb{F}\) and \(f\) in \(\mathbb{F^S}\), the product \(\lambda f \in \mathbb{F^S}\) is the function defined by \[ (\lambda f)(x) = \lambda f(x)\] for all \(x \in S.\)

Additive Identity and Inverse for \(\mathbb{F}^S\)

further notation: \(\mathbb{F}^S\)

  • the additive identity of \(\mathbb{F}^S\) is the function \(0 : \mathbb{S} \rightarrow \mathbb{F}\) defined by \[ 0(x) = 0 \] for all \(x \in S\)

  • For \(f \in \mathbb{F}^S\), the additive inverse of \(f\) is the function \(-f : \mathbb{S} \rightarrow \mathbb{F}\) defined by \[ (-f)(x) = -f(x) \] for all \(x \in S\)
